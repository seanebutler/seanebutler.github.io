This will ensure a safe and pain-free installation of LibreOffice.

Context

Make sure you are using a Windows PC before following these instructions.

Ingredients

  • A computer with Windows XP/Vista/7/8
  • 1.5 GB hard drive space
  • 256 MB of RAM
  • 1024x768 resolution with 256 colors

Steps

  1. Download the LibreOffice installer from the downloads page.

    The installer may take a few minutes to download depending on your internet connection speed.

  2. Find where you downloaded the installer and double-click on it to open it. Read the welcome message and click Next to continue(Figure 2)
  3. Select a Typical setup and click Next.

    Only select Custom setup if you have had experience installing software like this before.

  4. Select if you want to create a start link on the desktop and if you would like LibreOffice to start up whenever your system starts up. Click Install.
  5. Wait for the software to install then clickNext.

    If a dialog pops up asking if you want to install the software, click Yes.

  6. Click Finish and LibreOffice is now installed! (Figure 3)
