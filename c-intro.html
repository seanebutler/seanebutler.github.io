Introducing LibreOffice and its many uses.

LibreOffice is free office suite software that has many capabilities. This website will show you how to do the following:

  • Download and install LibreOffice
  • Use LibreOffice as an alternative to Microsoft Office
  • Write a letter in LibreOffice Writer
  • Create a simple spreadsheet in LibreOffice Calc
  • Create a Presentation in LibreOffice Impress
